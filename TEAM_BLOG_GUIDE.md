## For Team Members - How to Manage Blogs

### Step 1: Get Access
Ask the project administrator (Samir) to:
1. Go to [Sanity Manage Console](https://www.sanity.io/manage)
2. Select the "ASCEND Website" project
3. Navigate to **Settings** â†’ **Members**
4. Click **Invite members**
5. Enter your email address
6. Set your role as **Editor** or **Administrator**

### Step 2: Access the Blog Editor
1. Go to **https://ascend.sanity.studio/**
2. Sign in with your Google account (or the account you were invited with)
3. You'll see the blog management interface

### Step 3: Create a New Blog Post
1. Click **"Create"** â†’ **"Blog Post"**
2. Fill in the required fields:
   - **Title**: Your blog post title
   - **Slug**: Auto-generated URL (you can edit if needed)
   - **Excerpt**: Brief description (max 200 characters)
   - **Category**: Choose from dropdown
   - **Read Time**: Estimate in minutes
   - **Published At**: Set publication date

### Step 4: Write Your Content
1. **Content Editor**: Use the rich text editor to write your blog
   - Add headings (H1, H2, H3)
   - Format text (bold, italic)
   - Add links
   - Insert images
   - Create lists
   - Add blockquotes

2. **Featured Image**: 
   - Click "Select image" under Featured Image
   - Upload a new image or select from existing
   - Add alt text for accessibility

### Step 5: SEO Settings (Optional)
1. Expand the **"SEO Settings"** section
2. Add **Meta Description** (appears in search results)
3. Add **Keywords** (press Enter after each keyword)

### Step 6: Publish
1. Click **"Publish"** to make your blog live
2. The blog will automatically appear on the website
3. You can **Save as Draft** to work on it later

## Content Guidelines

### Image Requirements
- **Featured Image**: Recommended 1200x600 pixels
- **Content Images**: Any size (will be optimized automatically)
- **Alt Text**: Always add descriptive alt text for accessibility

### Writing Best Practices
- **Excerpts**: Keep under 200 characters
- **Headlines**: Use H2 and H3 for section breaks
- **Read Time**: Be accurate (roughly 200 words per minute)
- **Categories**: Choose the most relevant category

### Available Categories
- **Market Insights**: Industry trends, market analysis
- **M&A Strategy**: Mergers, acquisitions, strategic planning  
- **Fundraising**: Investment, capital raising, funding guides
- **Valuation**: Business valuation, pricing strategies
- **Nepal Business**: Local business news, Nepal-specific content

## Team Workflow

### Content Creation Process
1. **Draft**: Create and save as draft
2. **Review**: Share link with team for feedback
3. **Revise**: Make necessary changes
4. **Publish**: Go live when ready

### Collaboration Features
- **Comments**: Add internal comments for team feedback
- **Version History**: See all changes and revert if needed
- **Real-time**: Multiple people can work simultaneously

## Quick Tips

### Formatting Content
- Use **bold** for emphasis
- Use *italics* for subtle emphasis
- Create numbered lists for step-by-step guides
- Use bullet points for features/benefits
- Add blockquotes for important statements

### SEO Best Practices
- Include target keywords in title and content
- Write compelling meta descriptions
- Use descriptive headings (H2, H3)
- Add alt text to all images

### Image Best Practices
- Use high-quality, relevant images
- Optimize file sizes before upload
- Add descriptive alt text
- Use consistent image style/branding

## Troubleshooting

### Can't Access Studio?
- Check if you've been invited to the project
- Try logging out and back in
- Contact the project administrator

### Content Not Showing on Website?
- Make sure the post is **Published** (not Draft)
- Check that all required fields are filled
- Website updates may take a few minutes

### Need to Edit Published Content?
- Click on any published blog post
- Make your changes
- Click **"Update"** to save changes
- Changes appear on website immediately
